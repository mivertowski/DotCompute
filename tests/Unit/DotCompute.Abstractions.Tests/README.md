# DotCompute.Abstractions Tests

Comprehensive unit tests for DotCompute.Abstractions.

## Test Coverage Goal
- Target: 80% code coverage
- Focus: Unit tests with mocked dependencies
- Framework: xUnit + FluentAssertions + NSubstitute

## Test Categories
- Unit tests: Core functionality
- Integration tests: Component interaction
- Performance tests: Benchmarks and profiling

## Running Tests
```bash
dotnet test tests/Unit/DotCompute.Abstractions.Tests/
```
